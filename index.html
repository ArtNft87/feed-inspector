Corrections and task lists

Ready to apply

One row per product and field, with the old value and the value you chose. Apply these in the feed pipeline or sync, then refresh the portal.

Task lists, for what cannot be mapped

These cannot be derived from one decision, they need real content per product. Each list is a clean worklist to hand to a person, with product id, brand and category for context.

A missing or unregistered barcode, a missing specification value or a missing image cannot be guessed without fabricating data, so the tool lists them for a person rather than filling them in.
